# 🎬 短视频智能体

> **短视频内容不用愁** — 一站式短视频内容生成与多平台发布智能助手

本项目是一个基于 Flask + OpenAI 的 PWA（渐进式网页应用），提供四大核心模块，帮助你从零到一完成短视频内容策划、脚本生成、数字人制作方案输出，以及多平台一键发布文案生成。

## 功能概览

| 模块 | 功能 | 核心输出 |
|------|------|----------|
| 📝 爆款文案生成器 | 输入业务信息，按平台生成完整爆款文案脚本 | 标题、钩子、口播文案、对标拆解 |

## 快速开始（本地运行）

### 前提条件

- Python 3.9 或更高版本
- 一个有效的 [OpenAI API Key](https://platform.openai.com/api-keys)（或兼容 OpenAI 接口的第三方服务密钥）

### 1. 克隆项目

```bash
git clone https://github.com/liming020126-cpu/short-video-bao-kuan-generator.git
cd short-video-bao-kuan-generator
```

### 2. 创建虚拟环境并安装依赖

```bash
cd short_video_pwa
python3 -m venv .venv
source .venv/bin/activate        # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

### 3. 配置环境变量

```bash
cp .env.example .env
```

用文本编辑器打开 `.env`，填写你的 API Key：

```
OPENAI_API_KEY=sk-xxxxxxxxxxxxxxxxxxxxxxxx
OPENAI_MODEL=gpt-4o-mini
OPENAI_API_BASE=https://api.openai.com/v1
```

> 如果你使用国内第三方 OpenAI 兼容服务（如 DeepSeek、月之暗面、智谱等），将 `OPENAI_API_BASE` 改为对应服务的地址即可。

### 4. 启动应用

```bash
python app.py
```

在浏览器中打开 **http://localhost:8000** 即可使用。

## 环境变量说明

| 变量名 | 必填 | 默认值 | 说明 |
|--------|------|--------|------|
| `OPENAI_API_KEY` | ✅ 是 | — | OpenAI 或兼容服务的 API 密钥 |
| `OPENAI_MODEL` | 否 | `gpt-4o-mini` | 文本生成所用模型，建议 `gpt-4o-mini` 或 `gpt-4o` |
| `OPENAI_VISION_MODEL` | 否 | `gpt-4o` | 图片分析所用视觉模型（用于带货视频模块上传图片时） |
| `OPENAI_API_BASE` | 否 | `https://api.openai.com/v1` | API 基础地址，使用第三方服务时需修改 |
| `PORT` | 否 | `8000` | 本地运行端口 |

## 如何使用各模块

打开应用后，顶部有四个标签页，分别对应四个功能模块。

---

### 📝 模块一：爆款文案生成器

**适用场景**：你有一个生意/产品，想快速生成多平台爆款短视频脚本。

**操作步骤**：

1. 点击顶部 **「📝 爆款文案」** 标签
2. 在左侧表单填写：
   - **具体生意**：例如「宠物定制猫粮」「家政保洁服务」
   - **行业**：例如「宠物」「生活服务」
   - **受众客户群体**：例如「25-40岁养猫女性」
   - **核心卖点**：例如「天然无添加、定制配方、48小时发货」
   - **获客方式**：例如「短视频+私域」
   - **同行对标账号**：填写你想参考的竞品账号名称
   - **对标标题/爆款方向**：填写你看到的爆款标题或想要的内容方向
3. 选择 **风格**（薛辉 / 王泽旭 / 董十一 等）
4. 勾选目标 **平台**（抖音、小红书、视频号、快手）
5. 点击 **「生成文案」** 按钮
6. 右侧输出结果，每个平台一个完整板块，包含标题、钩子、口播文案、对标拆解
7. 点击 **「导出 TXT」** 保存结果到本地

---

### 🛍️ 模块二：带货视频生成器

**适用场景**：你有一个商品要带货，需要生成完整的视频制作方案，并获得数字人拍摄指令（可直接用于 HeyGen / D-ID / 硅基智能等工具）。

> 💡 **如果商品是服装类**，AI 会额外生成"虚拟试穿方案"，描述数字人如何穿上衣服进行展示。

**操作步骤**：

1. 点击顶部 **「🛍️ 带货视频」** 标签
2. **上传商品参考图**（可选，支持 JPG / PNG / WEBP，最大 10MB）
   - 点击上传区域或直接将图片拖拽进去
   - 上传后会显示图片预览
   - 如不上传图片，请在"商品描述"中详细填写商品外观和特点
3. 选择 **商品类目**（通用商品 / 服装上衣 / 美妆护肤 / 食品饮料 等）
4. 填写 **商品描述**：卖点、材质、特色功能等
5. 填写 **目标受众**、**价格区间**
6. 填写 **数字人形象描述**：例如「25岁年轻女性，温柔亲和气质，小清新风格」
7. 选择 **视频风格** 和目标 **平台**
8. 点击 **「生成带货方案」**
9. 右侧输出完整方案，包含：
   - 商品核心卖点分析
   - 完整带货口播脚本（≥300字）
   - 数字人拍摄指令（动作 / 表情 / 镜头节奏）
   - 场景与服装搭配建议
   - 服装类专属：虚拟试穿展示方案
   - 视频时间轴规划（0-3s / 3-8s / 8-15s / 15-30s / 30s+）
   - 各平台发布建议

---

### 🎙️ 模块三：口播脚本生成器

**适用场景**：你找到了一个效果很好的对标视频，想在保留爆款结构的前提下，改写出完全原创、贴合自身业务的口播脚本，同时获取数字人拍摄指令。

**操作步骤**：

1. 点击顶部 **「🎙️ 口播脚本」** 标签
2. 填写对标视频信息：
   - **对标视频链接**（可选，粘贴抖音/视频号/小红书链接）
   - **视频主题 / 选题方向**（必填）：例如「如何30天瘦10斤的健康减脂方法」
   - **视频内容简述**：描述对标视频的结构、钩子、核心内容（如无法提供链接，请尽量详细描述）
3. 填写你自己的信息：
   - **我的业务 / 领域**：例如「健康减脂餐定制服务」
   - **我的目标受众**：例如「25-40岁职场女性，想减肥但没时间运动」
4. 填写 **数字人形象描述**
5. 选择 **口播风格** 和目标 **平台**
6. 点击 **「生成口播方案」**
7. 右侧输出完整方案，包含：
   - 对标视频结构拆解（选题逻辑 / 钩子策略 / 情绪节奏）
   - 可复用爆款公式（2-3条）
   - 100% 原创改写口播脚本（≥400字）
   - 3个升级版开头钩子变体
   - 数字人拍摄指令（语速 / 停顿 / 表情 / 手势 / 镜头）
   - 场景模仿超越方案（模仿点 + 超越点）
   - 多平台适配版本

---

### 🚀 模块四：多平台一键发布

**适用场景**：你已经做好了视频，需要为每个平台生成专属的发布文案、话题标签、最佳发布时间等，直接复制粘贴就能发布。

**操作步骤**：

1. 点击顶部 **「🚀 一键发布」** 标签
2. 填写视频信息：
   - **视频标题 / 主题**（必填）：例如「三招教你在家做出餐厅级别的红烧肉」
   - **视频内容摘要**（必填）：描述视频的核心内容、结构、亮点
   - **内容类目**：从下拉菜单选择（美食教程 / 带货视频 / 知识科普 等）
   - **核心价值点 / 卖点**：视频最吸引用户的点
   - **目标受众**：例如「25-45岁家庭主妇、爱做饭的上班族」
   - **账号风格定位**：例如「家常美食博主，温暖治愈风」
3. 勾选目标 **发布平台**（抖音 / 小红书 / 视频号 / 快手 / B站）
4. 点击 **「生成发布方案」**
5. 右侧输出每个平台的完整发布方案，包含：
   - 3个备选标题（含不同钩子策略）
   - 正文文案（符合平台调性）
   - 话题标签（10-15个，含热门 + 精准长尾）
   - 封面文字建议
   - 最佳发布时段
   - 评论区互动引导话术
6. 复制对应平台的文案，前往各平台发布页直接粘贴发布

## 生产部署（Render）

本项目已配置好 `render.yaml`，可一键部署到 [Render](https://render.com)（免费套餐可用）。

1. 将项目 Fork 或推送到你的 GitHub 账号
2. 登录 [Render Dashboard](https://dashboard.render.com)
3. 点击 **New → Web Service**，连接你的 GitHub 仓库
4. Render 会自动识别 `render.yaml` 配置并完成部署设置
5. 在 Render 后台的 **Environment** 页面，添加以下环境变量：

   | 变量名 | 值 |
   |--------|-----|
   | `OPENAI_API_KEY` | 你的 OpenAI API Key |
   | `OPENAI_MODEL` | `gpt-4o-mini`（或其他模型） |
   | `OPENAI_API_BASE` | `https://api.openai.com/v1` |

6. 点击 **Deploy** 即可，部署完成后会获得一个公网 URL

## 常见问题

**Q：提示"缺少 OPENAI_API_KEY 环境变量"怎么办？**

A：请确认 `.env` 文件已正确创建并填写了 `OPENAI_API_KEY`。本地运行时确保你在 `short_video_pwa` 目录下启动 `python app.py`。

---

**Q：可以使用国内的 AI 服务（如 DeepSeek、月之暗面等）吗？**

A：可以。在 `.env` 中修改 `OPENAI_API_BASE` 为对应服务的 API 地址，并将 `OPENAI_MODEL` 改为该服务支持的模型名称即可。例如：

```
OPENAI_API_KEY=your-key-here
OPENAI_MODEL=deepseek-chat
OPENAI_API_BASE=https://api.deepseek.com/v1
```

注意：带货视频模块上传图片时，使用的是 `OPENAI_VISION_MODEL`（默认 `gpt-4o`），该功能需要所用服务支持视觉（vision）能力。

---

**Q：上传商品图片失败怎么办？**

A：请确认图片格式为 JPG / PNG / WEBP，文件大小不超过 10MB。GIF 格式不支持。

---

**Q：生成结果为空或报错怎么办？**

A：
1. 检查 API Key 是否有效且有余额
2. 检查网络是否能访问 OpenAI API（部分地区需要代理）
3. 如果使用第三方服务，确认 `OPENAI_API_BASE` 地址正确
4. 查看终端（或 Render 日志）中的错误信息获取详细原因

---

**Q：可以安装为手机 App 使用吗？**

A：可以。本项目是 PWA（渐进式网页应用），在手机浏览器中打开后，点击"添加到主屏幕"即可像 App 一样使用，无需应用商店安装。
